AlphaPlace [THE Alpha Server in 2020!]

What it is:

AlphaPlace is a server dedicated to our players nostalgia. The server was started on April, 7th, 2019. We have been using our current world for 224 days (As of 12/1/2020) and our player record is 31.

Info:

The server is running on Minecraft Beta v1.1_02, BUT WAIT! Beta 1.1_02 is the same as alpha 1.2.6 but with server side inventory which prevents normal players spawning in items and duping, which would ruin the server. The server's ip address is AlphaPlace.tk or AlphaPlace.net (both work).

The 2 popular ways of joining are: MultiMc and RetroWrapper We have a tutorial on how to use these methods on Youtube and our Website. You can read more on the development blog, or here about Minecraft in 2010.
